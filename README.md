# Pest-Pesticide Recommendation System 🌱

This is a simple prototype that helps farmers find the correct pesticide based on crop and pest input.

## Features
- Input crop and pest (e.g. "aphid on apple")
- Get pesticide recommendation
- Simple web interface

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
python app.py

3. Open browser:
http://127.0.0.1:5000
